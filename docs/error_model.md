# Error Model (JC0)

Purpose
- Define the stable error namespace used by BIOS and DOS.

Type
- jc_error_t is an unsigned 16-bit value.

Category ranges
- 0x0000: OK
- 0x0100-0x01FF: BOOT
- 0x0200-0x02FF: DISCOVERY
- 0x0300-0x03FF: BDT
- 0x0400-0x04FF: DEV
- 0x0500-0x05FF: FS
- 0x0600-0x06FF: EXEC
- 0x0700-0x07FF: MODE
- 0x0F00-0x0FFF: INTERNAL

Assignments
- 0x0000 JC_E_OK
- 0x0101 JC_E_BOOT_BSP_MISSING
- 0x0102 JC_E_BOOT_BSP_BAD_SIGNATURE
- 0x0103 JC_E_BOOT_DISCOVERY_MISSING
- 0x0104 JC_E_BOOT_UNSUPPORTED_PROFILE
- 0x0201 JC_E_DISCOVERY_BAD_SIGNATURE
- 0x0202 JC_E_DISCOVERY_BAD_VERSION
- 0x0203 JC_E_DISCOVERY_BAD_SIZE
- 0x0204 JC_E_DISCOVERY_BAD_POINTER
- 0x0301 JC_E_BDT_BAD_SIGNATURE
- 0x0302 JC_E_BDT_BAD_VERSION
- 0x0303 JC_E_BDT_BAD_SIZE
- 0x0304 JC_E_BDT_BAD_CHECKSUM
- 0x0401 JC_E_DEV_NOT_FOUND
- 0x0402 JC_E_DEV_UNSUPPORTED
- 0x0403 JC_E_DEV_IO_TIMEOUT
- 0x0404 JC_E_DEV_IO_ERROR
- 0x0501 JC_E_FS_BAD_SUPER
- 0x0502 JC_E_FS_BAD_DIR
- 0x0503 JC_E_FS_NO_SPACE
- 0x0504 JC_E_FS_NOT_FOUND
- 0x0601 JC_E_EXEC_BAD_MAGIC
- 0x0602 JC_E_EXEC_BAD_VERSION
- 0x0603 JC_E_EXEC_BAD_CRC
- 0x0604 JC_E_EXEC_UNSUPPORTED_TIER
- 0x0701 JC_E_MODE_INVALID_TARGET
- 0x0702 JC_E_MODE_NOT_PERMITTED
- 0x0703 JC_E_MODE_STACK_OVERFLOW
- 0x0704 JC_E_MODE_STACK_UNDERFLOW
- 0x0F01 JC_E_INTERNAL_ASSERT
- 0x0F02 JC_E_INTERNAL_UNREACHABLE

Invariants
- Codes are stable and never reused.
- New errors are appended within their category range.
